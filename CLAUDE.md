# Claude Code Adapter

Claude Code must read `AGENTS.md` first. `CLAUDE.md` is not the source of truth.

The source of truth for AI Logix is:

1. `AGENTS.md`
2. `planning/`
3. Active sprint folder
4. `docs/`
5. Existing code

Claude Code rules:

- Follow the active sprint.
- Do not use the conversation as primary memory.
- Do not redefine scope.
- Do not invent business rules.
- Validate against sprint acceptance.
- Record durable decisions in `planning/DECISIONS.md`.
- Record risks and missing answers in `planning/RISKS.md` and `planning/QUESTIONS.md`.
